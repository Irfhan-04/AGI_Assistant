# Demo Video Script for Hackathon Submission

## Video Structure (3-5 minutes)

### 0:00-0:30 - Introduction & Hook
**Visual**: Show the final automation working
**Narration**: 
> "Watch this AI complete my daily Excel report in 10 seconds - a task that normally takes me 2 minutes. This is The AGI Assistant, a local AI that watches, learns, and automates."

**Action**: 
- Show Excel report being created automatically
- Show time comparison (manual vs automated)

---

### 0:30-1:00 - Problem Statement
**Visual**: Show person doing repetitive manual work
**Narration**:
> "We all have repetitive tasks that waste our time. Creating reports, filling forms, organizing files. What if your computer could learn from watching you work?"

**Action**:
- Show examples of repetitive tasks
- Emphasize time waste

---

### 1:00-1:30 - Solution Overview
**Visual**: Show main application UI
**Narration**:
> "Meet The AGI Assistant - your personal AI that watches, learns, and automates. Everything runs 100% locally on your machine - no cloud, no data sharing, complete privacy."

**Action**:
- Show main window
- Highlight privacy features
- Show UI components

---

### 1:30-2:30 - Learning Phase (Round 1)
**Visual**: Recording session in action
**Narration**:
> "Here I am creating my daily Excel report manually. The AI watches everything - my clicks, my typing, my workflow. Notice it's capturing screenshots, recording audio, and tracking all my actions."

**Action**:
- Start recording
- Perform Excel task manually
- Show behind-the-scenes (screenshots being saved, events logged)
- Stop recording
- Show "Processing session..." message

**Repeat**: Do the same task 2 more times
**Narration**:
> "I do this same task again tomorrow, and again the next day. The AI is learning the pattern."

---

### 2:30-3:00 - Understanding Phase
**Visual**: Show pattern detection notification
**Narration**:
> "After three similar sessions, the AI recognizes the pattern. It understands this is a repetitive workflow I do every morning. It generates a structured workflow with all the steps."

**Action**:
- Show workflow card appearing
- Show workflow details (steps, confidence)
- Highlight "Pattern detected" message

---

### 3:00-4:00 - Automation Phase (Round 2)
**Visual**: Execute automation
**Narration**:
> "Now watch this - I just click 'Run Workflow' and the AI takes over. It executes every step automatically - opening Excel, filling cells, saving the file. Done in 10 seconds. That's an 80% time savings."

**Action**:
- Click "Run Now" button
- Show AI controlling mouse/keyboard
- Show Excel opening automatically
- Show cells being filled
- Show file being saved
- Show success message
- Show time comparison

---

### 4:00-4:30 - Privacy & Technical Details
**Visual**: Show settings/privacy tab
**Narration**:
> "Everything runs on your local machine. No cloud uploads. No data sharing. Your workflows stay yours. The AI uses Ollama with Phi-3.5 Mini, running entirely on your CPU."

**Action**:
- Show privacy tab
- Show storage usage
- Highlight "100% Local" messaging

---

### 4:30-5:00 - Closing & Call to Action
**Visual**: Show dashboard with multiple workflows
**Narration**:
> "The AGI Assistant - because your computer should work as hard as you do. Watch, learn, automate - all locally, all private."

**Action**:
- Show multiple learned workflows
- Show logo/title card
- End with contact/submission info

---

## Key Points to Emphasize

1. **Complete Pipeline**: Show all three phases (Observe → Understand → Automate)
2. **Privacy**: Emphasize 100% local processing
3. **Learning**: Show pattern detection across multiple sessions
4. **Automation**: Show actual task being automated
5. **Time Savings**: Quantify the benefit (e.g., "80% time savings")
6. **Professional**: Clean UI, smooth execution

## Tips for Recording

- Use screen recording software (OBS, Camtasia, etc.)
- Record in 1080p or higher
- Use clear narration or subtitles
- Show actual mouse movements and clicks
- Keep it under 5 minutes
- Edit out any mistakes or pauses
- Add captions/annotations for clarity

## Example Workflow to Demo

**Excel Daily Report**:
1. Open Excel
2. Create new workbook
3. Add headers: Date, Sales, Revenue
4. Fill in today's data
5. Save as "Report_YYYY-MM-DD.xlsx"
6. Close Excel

This is simple, visual, and easy to understand.
